= Chapter 17 Writer Project

Start the container using:

[source, shell]
----
podman-compose up
----

Let the terminal open. Just CTRL+C to exit and delete the containers.

Run with:

[source, shell]
----
# in one terminal
java -jar build/libs/chapter13-kafka-boot-6.0-SNAPSHOT.jar --app.sending.topic.name=Tom --app.receiving.topic.name=Evelyn --server.port=8080
# or
gradle bootRun --args='--spring.profiles.active=tom'

# in another terminal
java -jar build/libs/chapter13-kafka-boot-6.0-SNAPSHOT.jar --app.sending.topic.name=Evelyn --app.receiving.topic.name=Tom --server.port=8090
#or
gradle bootRun --args='--spring.profiles.active=evelyn'
----
